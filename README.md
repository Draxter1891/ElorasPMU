# 💄 Lip Blushing Landing Page – Elora’s PMU

A mobile-first, conversion-focused landing page built for **Lip Blushing in Delhi**, designed to match Elora’s premium aesthetic and drive consultation bookings.

---

## 🚀 Live Features

- Clean, premium UI inspired by Korean aesthetics  
- Mobile-first responsive design  
- Sticky CTA (WhatsApp & Call) for higher conversions  
- Lead capture form with minimal friction  
- SEO-focused structure targeting *“Lip Blushing in Delhi”*  
- Before/After storytelling, testimonials, and pricing sections  

---

## 🛠️ Tech Stack

- **React (Vite)**
- **Tailwind CSS**
- **Lucide Icons**
- Deployed on **Vercel** (recommended)

## 🎯 Approach

The project was built by focusing on **conversion over decoration**.  
The layout is structured to guide users from awareness → trust → action, using clear sections, strong CTAs, and minimal distractions. Special attention was given to mobile usability, fast decision-making, and aligning the design with Elora’s premium brand identity.

---

## 📈 Key Highlights

- Designed for **high-intent local traffic**
- Optimized for **quick booking actions**
- Clean component-based architecture for scalability
- Simple, readable, and maintainable codebase
